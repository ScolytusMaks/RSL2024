Справочники

Языки оригиналов статей

Общие сведения

Справочник "Языки оригиналов статей" предназначен для хранения перечня языков, использованных при написании гезетных статей.

Особенности справочника

Каждая запись справочника содержит два варианта названия языка - полное и сокращенное (библиографическое).

Справочник не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому полей "Полное наименование" и "Сокращенное наименование".

Картотека

Картотека "Территории распространения" не поддерживает иерархичность.

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.
