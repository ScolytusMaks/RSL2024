Справочники

Места изданий газет и МП

Общие сведения

Справочник "Места изданий" отражает действующую административно-территориальную организацию РФ. Справочник имеет иерархическую структуру, соответствующую действующей на текущий момент структуре административного подчинения географических объектов РФ.

Особенности справочника

Справочник не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому поля "Библиографическое наименование".

Поддержано до пяти уровней вложенности. На пятом уровне допускается создавать

Первый (самый верхний) уровень соответствует федеральным округам в структуре административного подчинения субъектов РФ. Каждая запись, описывающая федеральный округ, является группой (т.е. может содержать в себе подчиненные элементы). На этом же уровне хранится единственная негрупповая запись, описывающее неизвестное место издания.

Поддержано до пяти уровней вложенности. На пятом уровне допускается создавать только негрупповые записи, соответствующие населенному пункту (поселению).

Каждая запись должна характеризоваться административным статусом территории. Набор допустимых значений административного статуса зависит от уровня вложенности записи.

Картотека

Прямое редактирование (без формы-редактора) записей картотеки не предусмотрено. Разрешается перемещать записи из одной группы в другую способом "drag-and-drop".

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования (в режимах "по началу наименования" / "по произвольному фраменту наименования" / "по точному соответствию")
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Запись "неопределенного места издания" выделяется красной подсветкой.

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.

В форме отображаются все вышестоящие уровни подчинения с указанием их административного статуса. Для редактирования доступен только статус текущего уровня.

Поле "Наименование библиографическое" заполняется автоматически и для редактирования недоступно. На формирование библиографического наименования влияет значение полей:

  • Наименование сокращенное
  • флаг "Права субъекта РФ"
  • флаг "Не указывать административное подчинение".
