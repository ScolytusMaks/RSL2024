Справочники

Общие сведения

Справочники - это списки данных, которые используются в учете. Элемент справочника называется "запись". Справочники необходимы, чтобы исключить неоднозначную идентификацию каждого отдельно взятого элемента - как со стороны пользователя, так и со стороны программы. Это свойство справочников используется для организации учета и извлечения данных в отчеты.

Для идентификации каждая запись справочника содержит как минимум данные типа "ID" (идентификатор для программы) и данные типа "Наименование" (идентификатор для пользователя). Помимо этого элементы справочника могут содержать самую разную информацию - от развернутых сведений о сущности, отражением которой является элемент справочника (например, сведения о контрагенте) до правил обработки этой сущности (например, признак завершенности работы с элементом справочника).

Каждый справочник содержит относительно однотипную информацию. Перечень справочников определяется создателями программы. Задача наполнения справочников записями в большинстве случаев возлагается на пользователя.

Как правило, информация в отдельном элементе справочника изменяется нечасто, но количество записей в справочнике постоянно меняется в бОльшую сторону. В итоге количество записей одного справочника может быть значительным.

Картотеки справочников

Для получения доступа к записям справочнков пользователю предоставляется графический интерфейс типа картотека. Работа с картотекой, содержащей записи справочников, не отличается от работы с любой другой картотекой.

Эффективным приемом поиска необходимых записей в картотеках является наложение фильтра на записи. После наложения фильтра записи, не удовлетворяющие условиям фильтра, скрываются.

Условия, по которым выполняется фильтрация, могут быть достаточно сложными. Однако для решения повседневных задач чаще всего достаточно задать сочетание от одного до трех простых условий. Для этих целей дополнительные области многих картотек (включая картотеки справочников) снабжены панелью (областью) "Фильтр".

Реестр справочников

  • Отправители печатной продукции
  • Типы учетных документов
  • Места изданий газет и МП
  • Адреса редакций
  • Территории распространения (уровни изданий)
  • Языки изданий
  • Языки оригиналов статей
  • Тематики/целевое назначение/читательский адрес
  • Тематики статей
  • Библиотечно-библиографический классификатор
  • Географические рубрики
  • Авторы, персоналии
  • Серии статей
  • Постоянные организации
  • Постоянные организации - коллективы (основное деление)
  • Постоянные организации - структурные подразделения
  • Постоянные организации - временные подразделения
  • Временные организации
  • Временные организации - коллективы (основное деление)
  • Временные организации - структурные подразделения
