Справочники

Типы учетных документов

Общие сведения

Справочник "Типы учетных документов" предназначен для хранения информации о типах (видах) документов, сопровождающих поступающие экземпляры печатной продукции.

Особенности справочника

Справочник не содержит внешнего ключа.

Поле "Код", имеющееся в справочнике, предназначено для хранения аббривеатуры наименования типа документа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому полей "Код" и "Наименование".

Порядок следования записей в имеет значение: в учетном документе допустимые варианты типа документа будут представлены в том порядке, в каком они следуют в справочнике.

Картотека

Картотека "Типы учетных документов" не поддерживает иерархичность.

Управление порядком следования реализовано через контекстное меню, вызываемое кликом правой копки мыши на записи картотеки. Команды на измененение порядка следования записей доступны только при взведенном флаге "Пользовательская сортировка".

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.
