Справочники

Адреса редакций

Общие сведения

Справочник "Адреса редакций" предназначен для хранения неструктурированной адресной информации редакций газет в привязке к справочнику мест изданий.

Особенности справочника

Иерархическая часть справочника "Адреса редакций" является представлением справочника "Места изданий". Собственно справочник адресов имеет "плоскую" стрктуру.

Справочник не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому полей "Место издания" и строковому полю "Адрес".

Картотека

При перемещении селектора по иерархической части картотеки выполняется наложение фильтра на "плоское" представление (центральная область окна). Группы в списке адресов не отображаются никогда.

В справочник можно добавть только записи адресов, но не населенных пунктов. Иерархическая часть создается при редактировании справочника "Места изданий".

Особенности фильтрации.

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования географического объекта (в режимах "по началу наименования" / "по произвольному фраменту наименования" / "по точному соответствию")
  • одного или двух фрагментов строкового представления адреса (например - фрагменту почтового индекса и фрагменту наименования улицы).

При пользовании фильтром следует учитывать, что фильтр по наименованию географического объекта накладывается только на левую (иерархичную) часть формы. Фильтр по фрагментам адреса работает только для центральной ("плоской") области формы.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.

Выбор в форме-редакторе иного географического объекта приведет к автоматическому перемещению записи в группу выбранного объекта.
