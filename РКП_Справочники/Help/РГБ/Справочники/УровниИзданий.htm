Справочники

Территории распространения (уровни изданий)

Общие сведения

Справочник "Территории распространения (уровни изданий)" предназначен для хранения информации, классифицирующий издания по охвату территорий.

Кроме этого, каждая запись справочника может содержать сведения о плановом количестве обязательных экземпляров, предоставляемых изданием в РГБ.

Особенности справочника

Справочник не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому поля "Наименование".

Порядок следования записей в имеет значение: в тех отчетах, которые строятся с использованием этого справочника, данные будут представлены в том порядке, в каком они следуют в справочнике.

Картотека

Картотека "Территории распространения" не поддерживает иерархичность.

Управление порядком следования реализовано через контекстное меню, вызываемое кликом правой копки мыши на записи картотеки. Команды на измененение порядка следования записей доступны только при взведенном флаге "Пользовательская сортировка".

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.

В форме предусмотрена табличная часть, предназначенная для ввода данных о плановом количестве обязательных экземпляров. Строки в таблицу добавляются кликом по символу +, удаляются - кликом по символу x.

В таблице сопоставляется год, начиная с которого законодатель установил количество обязательных экземпляров для газет редактируемого уровня и количество обязательных экземпляров.

Предполагается, что законодатель не будет менять объявленное правило чаще, чем 1 раз в год. Таким образом, в таблице может быть столько строк, сколько раз произошло изменение правил.
