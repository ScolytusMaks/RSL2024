Справочники

Коллективные авторы

Общие сведения

Группа справочников "Коллективные авторы" используется при заполнении раздела "Первые сведения об ответственности" (подразделов "Постоянная организация", "Временная организация") карточки газетной статьи.

Группа включает в себя две подгруппы, по четыре справочника в каждой:

  • Группа "Постоянные организации":
    • Постоянные организации
    • Постоянные организации - коллективы (основное деление)
    • Постоянные организации - структурные подразделения
    • Постоянные организации - временные подразделения
  • Группа "Временные организации":
    • Временные организации
    • Временные организации - коллективы (основное деление)
    • Временные организации - структурные подразделения
    • Временные организации - временные подразделения.

Особенности справочников

Все справочники, входящие в группу "Коллективные авторы", похожи.

Справочники не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому поля "Наименование".

Картотека

Картотеки справочников группы "Коллективные авторы" не поддерживает иерархичность.

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.
