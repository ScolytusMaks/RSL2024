Справочники

Географические рубрики статей

Общие сведения

Справочник "Географические рубрики" используется при заполнении раздела "Географический рубрикатор" карточки газетной статьи.

Каждая запись справочника содержит два варианта наименования:

  • основная форма
  • малоупотребительная форма

Особенности справочника

Справочник не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому поля "Основная форма".

Картотека

Картотека "Географические рубрики" не поддерживает иерархичность.

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента основного наименования
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.

Поле "малоупотребительная форма" может не содержать значения.
