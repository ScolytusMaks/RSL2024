Реестр номеров

Картотека

Картотека "Номера (выпуски) газет и МП" предоставляет пользовательский интерфейс к записям, в каждой из которых сохранена информация об одном номере (выпуске).

Вызов картотеки предусмотрен только из формы "Карточка издания". При этом картотека всегда открывается с предустановленным фильтром на год выпуска. Пользовательского интерфейса для ввода быстрого фильтра в этой форме не предусмотрено.

По умолчанию сорировка выполняется сначала по столбцу "серия", затем - по столбцу "номер". Пользовательская сортировка возможна по любому столбцу (кликом лкм по заголовку столбца).

Допускается также менять порядок вывода, скрывать и изменять ширину столбцов. Все эти изменения сохраняются между сессиями для каждого пользователя индивидуально.

В эту область в верхнюю часть выводится информация о количестве экземпляров текущего номера, принятых на хранение. В нижней части выведены суммы ключевых показателей с учетом предустановленного фильтра записей.

Информационную панель можно скрывать. Для скрытия/восстановления используется управляющий элемент интерфейса "сплиттер" (выделен на скриншоте рамкой).

Добавление и удаление записей

Добавление новых записей в реестр возможно способами:

  • Нажатием клавиши "Insert"
  • Кликом по "кнопке" в правой части картотеки "Добавить".

Во всех случаях результатом будет вывод на экран формы-редактора записи Карточка номера.

Удаление записей:
  • Нажатием клавиши "Delete"
  • Кликом по "кнопке" в правой части картотеки "Удалить".
Во всех случаях программа запросит подтверждение действия.

Нельзя удалять записи, для которых введено количество. Если удаление записи номера всё же необходимо, то сначала нужно удалить данные о количестве принятых экземпляров.

Удалить количество принятых экземпляров невозможно, если они уже отражены в учетном документе.

Форма-редактор

Редактирование записей выполняется в форме - редакторе записи.

Порядок работы с формой описан в разделе Карточка номера.
