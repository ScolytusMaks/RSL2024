Единицы хранения

Картотека

Картотека "Единицы хранения" предоставляет пользовательский интерфейс к записям, где фиксируются факты получения и принятие на хранение обязательных экземпляров газет/МПГ. Записи в таблице создаются в привязке к номеру (выпуску) газеты/МПГ. Каждая запись содержит информацию:

  • о количестве принятых экземпляров
  • о дате фактического поступления
  • произвольное примечание
  • технические сведения об авторстве и времени создания записи.

Картотека "Единицы хранения" не предусматривает возможностей ввода и удаления записей. Возможно только редактирование записей в том случае, если администратором системы предоставлены соответствующие права. Создание записей выполняется посредством интерфейса "Номер".

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагменту названия издания
  • году выпуска
  • фрагменту номера
  • авторства текущего пользователя.

Форма-редактор

Редактирование записей выполняется в форме - редакторе записи.

Форма предоставляет доступ к редактированию значимых полей. Для просмотра доступны учетная цена номера (выпуска) и некоторые техничесие сведения о записи.

Для сохранения измененной записи необходимо, чтобы поле "Дата поступления" было заполнено, количество экземпляров должно быть ненулевым неотрицательным значением.
