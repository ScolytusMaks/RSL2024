Учет фактических поступлений газет и МП

Учетные документы

Общие сведения

Учетные документы - это любые финансовые (хозяйственные) документы, подтверждающие факт передачи обязательных экземпляров газет/МПГ отправителем печатной продукции Российской государственной библиотеке.

Функциональный блок регистрации входящих учетных документов состоит из картотеки (реестра документов) и формы ввода / редактирования данных конкретного документа.

Картотека

Добавление новых записей

Добавление новых записей в картотеку возможно следующими способами:

  • Нажатием клавиши "Insert"
  • Кликом по "кнопкам" в правой части картотеки "Добавить" или "Дублировать"
  • Вызовом контекстного меню (клик ПКМ) в области вывода записей и выбором команд "Добавить" или "Дублировать".

Во всех случаях результатом будет вывод на экран формы-редактора записи "Учетный документ".

При дублировании записи содержимое табличной части не копируется. Во вновь созданной записи будут заполнены только реквизиты документа (отправитель, номер, дата) и контрольные значения общей суммы и общего количества.

Удаление записей

Допускается удаление только тех документов, у которых не заполнена табличная часть.

Удаление документов с заполненной табличной частью с игнорированием проверки ссылочной целостности приведет к разрушению внутренних связей данных.

Поиск записей

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • признака завершенности работы с документом
  • авторства текущего пользователя.

По умолчанию картотека открывается со сброшенным флагом "Показать завершенные" и взведенным флагом "Только мои". Т.е. по умолчанию фильтр настрое так, что пользователь видит только те записи, которые когда-то создал он сам и при этом документы не содержат признака завершеннсти работы с ними.

По любому из столбцов, выведенных в пользовательский интерфейс, возможна сортировка и фильтрация записей. Подробно приемы работы с картотекой описаны в разделе Принципы работы с картотеками.

Форма-редактор

Порядок заполнения формы

  • Выбрать тип учетного документа из выпадающего списка.

    Содержимое списка управляется справочником Типы учетных документов. Для редактирования справочника администратором системы должны быть предоставлены соответствующие права. В списке не будет типов документов, помеченных как "устаревшие".

  • Ввестии номер и дату составления учетного документа.
  • Выбрать отправителя из справочника Отправители.

    Для открытия справочника кликните кнопку обзора "..." в поле "Отправитель" или "ИНН". Для поиска в справочнике введите фрагмент наименования в поле "Отправитель" или фрагмент ИНН в поле "ИНН", после чего нажмите клавишу "Enter" или кликните кнопку обзора. Справочник откроется с фильтром на введенные значения. После открытия условия фильтрации доступны для изменения.

    Если сведения об отправителе отсутствуют в справочнике, их можно добавить, не покидая диалог выбора. Добавление возможно, если администратором системы предоставлены соответствующие права.

  • Ввестии контрольные значения итоговой суммы и итогового количества экземпляров печатной продукции.

    Ввод этих значений не обязателен. Наличие контрольных сумм помогает контролировать корректность данных при заполнении табличной части документа.

  • Добавить строку в документ.

    Строки в табличную часть добавляются кликом по символу +. Допускается создавать сразу несколько незаполненных строк.

  • Заполнить данные в строке документа.

    Данные в строке документа считаются корректными, если выполнен выбор ранее введенного количества поступивших единиц хранения. Выбор данных из картотеки поступлений ("Единицы хранения") возможен в колонке "Количество". Для выбора необходимо установить селектор в клетку на пересечении строки и столбца "Количество", нажать клавишу "Enter" или кликнуть кнопку обзора "..." в клетке.

    В целях уменьшения количества видимых записей при выбора количества используйте клетки в столбцах "Издание (газета)", "Год выпуска", "Номер (выпуск)" для формирования фильтра отбора. Например, для ввода количества поступивших номеров газеты "Известия народных депутатов" за 2024 год, номера 12-18 введите:

    в колонку "Издание (газета)" -> "Изве"

    в колонку "Год выпуска" -> "2024"

    в колонку "Номер (выпуск)" -> "1".

    При "проваливании" в клетку "Количество" картотека откроется с фильтром. Значения для фильтрации отобразятся на правой панели (область "Фильтр"), здесь их также можно изменить.

    До тех пор, пока количество не выбрано, текст в клетках "Издание (газета)", "Год выпуска", "Номер (выпуск)" выводится серым курсивом. После выбора - обычным шрифтом.

    После выбора количества использование клеток "Издание (газета)", "Год выпуска", "Номер (выпуск)" для задания фильтра невозможно.

    После того, как запись "Единицы хранения" выбрана (включена в состав редактируемого учетного документа), она становится недоступной для выбора в текущий и иные учетные документы.

  • Ввести цену номера.

    Цена номера вводится в клетку в колонке "Цена" без вызова дополнительных диалогов. Значение цены привязано к записи номера газеты (не к записи поступления!). Если фактическое поступление экземпляров одного номера было не одномоментным, то в картотеке "Единицы хранения" может оказаться несколько записей, соответствующих одной строке печатной формы документа. В этом случае цена вводится/изменяется в любой из соответствующих строк экранной формы.

  • Удалить лишние строки в табличной части.

    Строки удаляются кликом по символу x.

    При удалении строки запись в картотеке "Единицы хранения" не удаляется, но становится доступной для включения в текущий или иной учетный документ.

  • Выполнить контроль соответствия сумм и количества в табличной части учетного документа контрольным значениям.

    Разница между контрольными и введенными в табличную часть данными пересчитывается постоянно в процессе ввода. При ненулевых результатах значения разницы выводятся красным цветом. При совпадении выводятся значения соответственно "0" и "0.00" зеленым цветом.

  • Ввести произвольный комментарий к документу (при необходимости).
  • После того, как принято решение об окончании работы с учетным документом, взвести флаг "Завершен".

    После взвода флага "Завершен" учетный документ перестанет отображаться в картотеке (открытой с фильтром, задаваемым по умолчанию).

    Решение об окончании работы с документом не зависит от факта расхождения/совпадения контрольных значений и сумм значений в позициях табличной части.
