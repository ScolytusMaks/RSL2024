Карточка издания

Общие сведения

Форма "Карточка издания" предоставляет пользовательский интерфейс к данным ОДНОЙ записи реестра газет. В форму выведено ограниченное число полей - только те, которые позволяют однозначно идентифицировать издание.

Для пользователей, имеющих компетенции "Руководитель отдела ППИ", "Сотрудник отдела ППИ" доступ к иным сведениям об издании не предусмотрен.

Поля и управляющие элементы

Для идентификации издания на форму выведены сведения:

  • N гос. регистрации
  • ISSN
  • Тип издания
  • Газета отсылки (если издание имеет признак отсылки)
  • Название (основное)
  • Название на иностранном и национальном языках
  • Подзаголовок
  • Места издания
Флаги (недоступные для редактирования):
  • Прекращено
  • Отсылка
На правой панели, содержащей стандартные управляющие элементы (кн. "Закрыть", "Помощь"), выведены:
  • кнопки перемещения на соседние записи
  • флаг "Показывать только неполученные комплекты".
    При взведенном флаге скрывается информация о одах, за которые получен полный комплект номеров.

Регистрация года выпуска

Сведения о поступивших на хранение и регистрацию номерах (выпусках) издания сгруппированы по годам выпуска. Одновременно можно получить доступ только к номерам одного издания за один год.

Область формы, предназначенная для управления данными о номерах (выпусках), реализована как таблица:

Чтобы ввести данные о номерах, выпущенных за год, которого нет в таблице, необходимо добавить новую строку. Строки в таблицу добавляются кликом по символу +, удаляются - кликом по символу x.

Удалить строку можно только при отсутствии введенных номеров за год, указанный в строке.

При создании новой записи о годе выпуска поле "Год" в форме-редакторе записи доступно для изменения:

По умолчанию программа подставляет год, следующий за последним введенным. При полном отсутствии ранее введенных данных - текущий год. В этот момент значение года можно измениить (при необходимости).

При входе в поле год (одиночный клик ЛКМ по кнопке обзора или двойной по клетке) форма "Выпуск издания в течение года" доступна для просмотра. Изменение года в этом режиме невозможно. Ввод данных о количестве поступивших экземпляров

Ввод данных о количестве поступивших экземпляров выполняется после входа в клетку в колонке "Номера".

На экран выводится форма Реестр номеров. Ввод данных о количестве поступивших экземпляров возможен только в контексте номера.
