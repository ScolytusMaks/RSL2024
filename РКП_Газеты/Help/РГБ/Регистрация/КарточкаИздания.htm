Регистрация газет и МП

Карточка издания

Общие сведения

Форма "Карточка издания" предоставляет пользовательский интерфейс к данным ОДНОЙ записи реестра газет.

Кроме этого, форма предоставляет пользовательский интерфейс к данным:

  • Реестр номеров (выпущенных и зарегистрированных)
  • Накопленной статистической информации в разрезе года выпуска номеров
  • Накопленной статистической информации в разрезе года регистрации номеров.

Следует отметить, что иного способа просмотра / редактирования перечисленной информации для пользователей с обычными правами не предусмотрено.

Области формы

Форма состоит из нескольких областей. Часть этих областей доступна постоянно, часть расположена на "закладках".

Область идентификации

Постоянно видимая область. Содержит сведения, позволяющие однозначно идентифицировать издание.

Поля ввода/вывода, размещенные в этой области, предполагают ввод только с клавиатуры или из буфера обмена.

Набор чекбоксов ("флагов"), описывающих статус записи, будет различный в зависимости от набора компетенций, указанных в правах пользователя.

Закладка "Место выхода и издатели"

Закладка содержит информацию об учредителе (группе учредителей), месте (местах) издания, адресную и иную контактную информацию издательства, сведения о редакторе (главном редакторе) издания.

Особенности редактирования полей ввода/вывода закладки "Место выхода и издатели"
  • Сведения об учредителе (группе учредителей).
    При взводе/сбросе флага "группа" проверка правильности склонения в поле ввода/вывода НЕ выполняется.
  • Места издания.
    Поскольку газета (МПГ) может выпускаться в нескольких населенных пунктах, область сведений о местах издания представляет собой табличную структуру. Новые строки добавляются клавишей "Insert", кликом ЛКМ на пиктограмме + или с помощью контекстного меню. Удаление - клавишей "Delete", кликом ЛКМ по пиктограмме x или с помощью контекстного меню.
    Выбор значения выполняется из справочника мест изданий. Окно справочника вызывется нажатием клавиши "Enter" при нахождении селектора в поле ввода/выводе или кликом ЛКМ по кнопке обзора.
    Поле допускает прямой ввод значения с клавиатуры. При таком сценарии ввода справочник мест изданий откроется с фильтром по фрагменту наименования географического объекта. Этот фильтр можно сбросить уже в окне справочника.
    В случае, если выбор из справочника уже был выполнен ранее, справочник откроется с фильтром на ранее выбранное значение.
    Если на момент вызова справочника в поле ничего не введено с клавиатуры и не выбрано из справочника, то окно выбора места издания открывается без фильтра. Позиционирование при этом выполняется на последнем выбранном значении.
    Порядок перечисления мест издания важен. Его можно изменить кликом ЛКМ по пиктограммам, нажатием клавиш "Alt+PgUp", "Alt+PgDn" или с помощью контекстного меню.
  • Редакция.
    Значение выбирается из справочника.
    В случае, если выбор редакции был выполнен ранее, фильтр на справочник не накладывается, но позиционирование выполняется на ранее выбранном значении.
    Поле допускает прямой ввод значения с клавиатуры. При таком сценарии ввода справочник адресов откроется с фильтром по фрагменту адреса - в том числе и в случае, если ранее значение уже было выбрано.

Поля "Учредитель (группа)", "Места издания", "Редакция", "(Главный) редактор" поддерживают сохранение истории изменения значений.
Для поля "Места издания" сохранение истории изменений поддерживается только для первого значения в списке мест изданий.
Форма просмотра / редактирования истории изменений вызывается кликом ЛКМ по пикторамме справа от поля ввода/вывода.

Закладка "Классификация"

Классификация изданий предусмотрена по:
  • Типу
  • Периодичности выпуска
  • Территории распространения
  • Тематике
  • Целевому назначению (читательскому адресу)
  • Языку.
Особенности редактирования полей ввода/вывода закладки "Классификация"

Все поля на этой закладке допускают ввод только предопределенных значений.

  • Тип издания.
    Допускается выбор только значений "Газета" или "МПГ" из выпадающего списка.
  • Периодичность выпуска.
    Периодичность определяется значением пары полей - "за какой период" и "количество выпусков". Поля взаимосвязаны, программа не допускает ввод недопустимых сочетаний. Значения выбираются из выпадающего списка.
  • Территория распространения.
    Значение выбирается из справочника.
    Поле допускает прямой ввод значения с клавиатуры. В этом случае справочник "Территория распространения" откроется с фильтром по фрагменту наименования. Без фильтра справочник откроется только в том случае, если в поле ввода/вывод нет никаких значений.
    Очистка поля выполняется нажатием клавиши "Delete".
  • Поля "Тематики", "Целевые назначения", "Языки издания".
    Области сведений о тематиках, целевых назначениях, языках издания организованы как табличные структуры, допускающие ввод нескольких значений. Новые строки добавляются клавишей "Insert", кликом ЛКМ на пиктограмме + или с помощью контекстного меню. Удаление - клавишей "Delete", кликом ЛКМ по пиктограмме x или с помощью контекстного меню.
    Выбор значения выполняется из справочников. Окно справочника вызывется нажатием клавиши "Enter" при нахождении селектора в поле ввода/выводе или кликом ЛКМ по кнопке обзора.
    Поля допускает прямой ввод значения с клавиатуры. При таком сценарии ввода справочники открываются с фильтром по фрагменту наименования, введенного в поле. Этот фильтр можно сбросить уже в окне справочника.
    В случае, если выбор из справочника уже был выполнен ранее, справочник откроется с фильтром на ранее выбранное значение.
    Если на момент вызова справочника в поле ничего не введено с клавиатуры и не выбрано из справочника, то окно выбора места издания открывается без фильтра.
    Порядок перечисления значений важен. Его можно изменить кликом ЛКМ по пиктограммам, нажатием клавиш "Alt+PgUp", "Alt+PgDn" или с помощью контекстного меню.
    Допускается выбор устаревших значений из справочников "Тематики" и "Целевые назначения". В случае, если выбрано устаревшее значение или признак "Устаревшее" был установлен ранее выбранному элементу справочника, то напротив строки с устаревшим значением выводится символ "?".

Закладка "Выход и закрытие"

При открытии карточки издания эта закладка устанавливается текущей (активной).

Закладка содержит информацию о выпуске самого первого номера издания, сведения об общем количестве полученных/неполученных номеров по годам выпуска, сведения о последнем полученном номере по годам выпуска.

Редактирование информации о номерах возможно во вспомогательном окне, вызываемом нажатием клавиши "Enter" или кликом ЛКМ по клетке вывода количества номеров. Порядок работы с формой описан в статье Реестр номеров.

Кнопка обзора в столбце "ЛГС" доступна в случае наличия в базе данных номеров, описанных пользователями с компетенциями "сотрудник группы ЛГС".

Секция, содержащая сведения о последнем выпущенном номере, выводится только при наличии метки "последний номер" в записях номеров. В связи с тем, что накопление данных происходило длительное время в различных реализациях пользовательского интерфейса, эта метка для прекращенных изданий может присутствовать не всегда.

Особенности редактирования информации на закладке "Выход и закрытие"

Сохранение даты выпуска первого номера возможно с точностью "День", "Месяц" или "Год". Для точности "День" допускается прямой ввод даты в поле "Дата выхода". Клик ЛКМ по кнопке обзора вызывет стандартный диалог "Календарь".
Для точности "Месяц" или "Год" прямой ввод невозможен, значение вводится только с помощью вспомогательных диалоговых окон.

Область сведений о выпусках организована как табличная структура, каждая строка которой соответствует одному году выпуска.

Флаг "Показывать только неполученные комплекты" в области управления формой позволяет скрыть информацию о годах, для которых работа по регистрации номеров завершена.

Добавление очередного года выпуска возможно только кликом ЛКМ по пиктограмме +. Изменение года выпуска (если год, предложенный по умолчанию, не соответствует реальной информации) возможно только в момент добавления.

Удаление строки возможно только в том случае, если для удаляемого года отсутствует информация о номерах (выпусках) за этот год. Удаление строки выполняется кликом ЛКМ по пиктограмме x.

Форма редактирования сведений о годе выпуска вызывается нажатием клавиши "Enter", двойным кликом ЛКМ по клетке в столбце "Год выпуска" или одиночным кликом ЛКМ по кнопке обзора в этом же столбце.
Работа с формой "Выпуск издания в течение года" описана в отдельной статье.

Поле "Комментарий" позволяет вводить произольный текст.

Закладка "Поступления"

Область сведений о фактических поступлениях организована как табличная структура, каждая строка которой соответствует одному году фактических поступлений (по дате регистрации номера).

Данные на этой закладке недоступны для редактирования. Для номеров, фактически принятых (зарегистрированных) до 2008 года, информация может отсутствавать.

Форма просмотра сведений о годе поступления вызывается нажатием клавиши "Enter", двойным кликом ЛКМ по клетке в столбце "Год принятия на хранение" или одиночным кликом ЛКМ по кнопке обзора в этом же столбце.
Работа с формой "Годовая статистика поступлений" описана в отдельной статье.

Закладка "Библиогр. запись"

Закладка содержит поля, значения которых используются в дальнейшем для формирования ежегодной "Летописи периодических и продолжающихся изданий".

Поле "Примечание библиографа" предполагает ввод комментрия к библиографической записи.

Особенности редактирования информации на закладке "Библиографическая запись"

После ввода значения в поле "Год летописи" поля "Заголовок" и "Описание" заполняются автоматически.
Автоматически сформированная библиографическая запись и заголовок записи доступны ручной правки. В поле "Описание" поддержан перевод каретки (ввод с новой строки).
Клик ЛКМ по кнопке "Обновить" удаляет правки, сделанные вручную.

Флаг "Исправлено библиографом" автоматически НЕ ВЗВОДИТСЯ.
