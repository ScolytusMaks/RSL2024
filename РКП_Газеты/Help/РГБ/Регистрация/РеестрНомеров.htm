Регистрация газет и МП

Реестр номеров

Общие сведения

Форма "Номера (выпуски) газет и МП" предоставляет пользовательский интерфейс к записям, в каждой из которых сохранена информация об одном номере (выпуске).

Вызов реестра предусмотрен только из формы Карточка издания. При этом предусмотрен доступ только к номерам конкретного года выпуска.

Записи в реестре отсортированы сначала по столбцу "серия", затем - по столбцу "номер".

Допускается менять порядок вывода, скрывать и изменять ширину столбцов. Все перечисленные команды доступны через контекстное меню, вызываемое кликом ПКМ в области заголовков столбцов. Все эти изменения сохраняются между сессиями для каждого пользователя индивидуально.

Восстановление раскладки и сортировки, предусмотренных по умолчанию, возможно также через контекстное меню.

При находении селектора в области списка номеров шрифт в текущей строке дополнительно выделяется начертанием.

Добавление и удаление записей

Добавление новых записей в реестр возможно способами:
  • Нажатием клавиши "Insert" (при нахождении селектора в области списка номеров);
  • Кликом по кнопке "Добавить" в правой части реестра.

После подачи команды на добавление предлагается выбрасть режим добавления - "по одному номеру" или "списком".
При выбора способа "по одному номеру" на экран выводится форма-редактор записи "Номер". Порядок работы с формой описан в отдельной статье.
При выбора способа "списком" на экран выводится форма, позволяющая ввести информацию в режиме "с номера ... по номер ...". Порядок работы с формой описан в отдельной статье.

Удаление записей возможно:
  • Нажатием клавиши "Delete" (при нахождении селектора в области списка номеров);
    В режиме прямого редактирования нажатие клавиши "Delete" инициирует процесс удаления записи только при нахождении селектора в недоступной для редактирования клетке.
  • Кликом по кнопке "Удалить" в правой части реестра.
Во всех случаях программа запросит подтверждение действия.

Нельзя удалять записи, для которых введено количество принятых экземпляров. Если удаление записи номера всё же необходимо, то данные о количестве принятых экземпляров необходимо удалить до момента удаления сведений о номере.

Удалить количество принятых экземпляров невозможно, если они уже отражены в учетном документе.

Удаление группы записей.

Для удаления группы записей необходимо выделить горизонтальный или прямоугольный блок в области списка номеров.
Для выделения горизонтального блока необходимо удерживая клавишу "Ctrl", зажать левую кнопку мыши и провести указателем в вертикальном направлении.
Для выделения прямоугольного блока необходимо сделать то же самое, но удерживая клавишу "Shift".
При удержанной клавише "Shift" выделение прямоугольного блока возможно также с помощью клавишей перемещения ("стрелки").

После выделения блока команда на удаление инициируется нажатием клавиши "Delete" или кликом ЛКМ по кнопке "Удалить".

Редактирование записей

Предусмотрено два режима редактирования записей:

  • В форме-редакторе записи "Номер";
  • Режим редактирования напрямую в реестре.
    Этот режим включается взводом флага в области управления формой.
    В режиме прямого редактирования возможно изменение только части полей, которые выведены в реестр. Доступные для редактирования поля подсвечены.

Информация о поступлениях

Информация о поступлениях экземпляров номера выводится в область управления формой. По умолчанию информационная часть области управления скрыта. Раскрывается область кликом по пиктограмме.
