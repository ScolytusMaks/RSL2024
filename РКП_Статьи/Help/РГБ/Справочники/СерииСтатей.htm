Справочники

Серии статей

Общие сведения

Справочник "Серии статей" используется при заполнении раздела "Область серии" карточки газетной статьи.

Особенности справочника

Каждая запись справочника связвает серию и издание. Для одного издания может существовать множество серий. В разных изданиях могут существовать одноименные серии.

Справочник не содержит внешнего ключа.

Контроль повторяющихся элементов справочника - "мягкий". При попытке сохранить в справочнике новую запись система выполнит анализ содержимого справочника на наличие "похожих" записей и потребует от пользователя подтверждения действия. Поиск "похожих" записей выполняется по содержимому полей "Издание" и "Наименование серии".

Картотека

Картотека "Серии статей" не поддерживает иерархичность.

Фильтр, встроенный в пользовательский интерфейс, позволяет отбирать записи по комбинации:

  • фрагмента наименования серии
  • признака устаревшей записи.

Записи, помеченные признаком "устаревшие", выделяются в картотеке серым курсивом (в случае, если они попадают под пользовательский фильтр).

Форма-редактор

Редактирование элемента справочника выполняется в форме - редакторе записи.

При создании записи необходимо указать издание, в котором регистрируется новая серия. Без выбора издания запись не создается.

При вызове справочника из карточки статьи создавать записи можно, если администратор системы предоставил соответствующие права. При создании новой записи в таком режиме поле "Название газеты" заполняется автоматически.

Название газеты в этом справочнике выводится так, как оно зарегистрировано специалистами отдела РГП (в ЛГС название газеты может отличаться).
